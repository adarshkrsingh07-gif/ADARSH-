## Habit Tracker with Insights

A simple *Python habit tracker* that helps you monitor your daily habits, visualize your progress, and understand your consistency patterns.

# Features

- Add and log daily habits  
- Track completion rates, streaks.
- Get insights on your best and worst habits  
- Visualize your growth using matplotlib graphs  
- Data stored locally in `data/habits.csv`

# Tech Stack

- **pandas** → data handling and analysis  
- **numpy** → calculations
- **matplotlib** → visualizations  
- **csv / file I/O** → persistent storage

---

# Installation & Setup

1. Open a terminal in the project folder  
2. Install dependencies: *pip install -r requirements.txt*
3. Run *python src/main.py*

# Usage

1. Add a habit
2. The program stores it in data/habits.csv
3. Mark completion - Update status (1 for done, 0 for missed)
4. See insights - See streaks, completion percentages, and top habits
5. Visualize - View charts of your consistency and progress over time

*Dummy data is provided for testing.*
